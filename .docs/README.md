# Docs (interno)

Este diretorio contem anotacoes internas do projeto (produto, arquitetura, release, etc).

## Play Console: Categoria e Tags

Objetivo: posicionar o Take Note como um app de notas simples, rapido e offline-first.

## Play Console: Descricoes (PT-BR e EN)

Notas:
- Evitar prometer sync/cloud (nao existe hoje).
- Reforcar offline-first, velocidade e blocos (listas/checklists/imagens).
- Manter a breve descricao curta (estilo slogan).

### Breve descricao (PT-BR)

Opcao A:
> Notas offline, rapidas e organizadas por blocos.

Opcao B:
> Crie notas, listas e checklists. Tudo offline.

### Descricao completa (PT-BR)

Take Note e um app de notas minimalista e offline-first, feito para capturar ideias sem friccao.

Com um editor em blocos, voce pode combinar:
- Texto
- Titulos e subtitulos
- Citacoes
- Listas e checklists
- Imagens

Recursos:
- Funciona offline (armazenamento local)
- Organizacao por categorias
- Arquivar e desarquivar notas
- Interface leve e rapida

Privacidade:
- Suas notas ficam no seu dispositivo.

### Breve descricao (EN)

Option A:
> Fast, offline-first notes with blocks.

Option B:
> Notes, lists, and checklists. Offline-first.

### Full description (EN)

Take Note is a minimal, offline-first notes app built for fast capture.

With a block-based editor, you can mix:
- Text
- Titles and subtitles
- Quotes
- Lists and checklists
- Images

Features:
- Works offline (local storage)
- Organize with categories
- Archive and restore notes
- Lightweight, fast UI

Privacy:
- Your notes stay on your device.

### Categoria (App category)

- Categoria: **Produtividade**

Justificativa:
- O core do app e captura/organizacao de notas, listas e checklists.
- E a categoria mais intuitiva para "notes app" no Google Play.

### Tags (Store listing tags)

Sugestao (manter em ingles para alcance em EUA/CA e consistencia no Play):

- `Notes`
- `Checklist`
- `To-do list`
- `Offline`
- `Productivity`

Observacoes:
- Se o Console limitar a quantidade de tags, priorizar nesta ordem: `Notes`, `Checklist`, `To-do list`, `Offline`, `Productivity`.
- Evitar tags que prometem features nao existentes (ex: `Sync`, `AI`, `Collaboration`).
